Novo Projeto: The Sixth Man 🏀

Criei uma aplicação para dar luz aos jogadores que mudam o destino das partidas da NBA saindo do banco de reservas.

Destaques técnicos do projeto:
✅ Arquitetura Híbrida: Combinei a SDK da balldontlie (TypeScript) com os endpoints oficiais da NBA para criar uma base de dados rica e visual.
✅ Lógica de Negócio: Desenvolvi um algoritmo de filtragem que isola a performance de reservas, calculando métricas de eficiência que plataformas comuns ignoram.
✅ Foco em UX: Implementei padrões de design modernos, skeleton screens para carregamento e uma interface totalmente responsiva.

Este projeto foi um desafio incrível de manipulação de dados reais e integração de múltiplas APIs. Confira o código aqui: [Link do GitHub]
